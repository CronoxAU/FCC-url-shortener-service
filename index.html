URL Shortener Service

A FreeCodeCamp Project by Christopher McLean

Input for shortener creation

A string containing the URL to the page a shortened URL is required for

Output

A JSON object the supplied URL (original_url) and a shortened URL (short_url) that will redirect to this location.

On invalid input (ie a string which is not a valid URL), a JSON object will be returned with both properties set to null.

Usage examples

Input Output
{ "original_url":, "short_url": }
{ "original_url":, "short_url": }
redirect to ***

Input for user redirection

A generated shortened URL

Output

The user will be redirected to the original URL
